# Practice3 - RemoteGalleryService

Вебсервіс віддаленого доступу до персоналізованої галереї зображень. Проєкт побудовано на базі розділеної клієнт-серверної архітектури (SPA) із використанням сучасного стеку технологій для фронтенду та бекенду.

## Технологічний стек

### Frontend (папка /practice)
* React (бібліотека для побудови декларативних інтерфейсів)
* TypeScript (строга типізація для надійності коду)
* Vite (швидкий інструмент збірки та локальний сервер розробки)
* React Context API (глобальне керування станом теми оформлення)

### Backend (корінь проєкту)
* C# .NET Web API (обробка HTTP-запитів, логіка контролерів)
* ASP.NET Core Static Files (збереження та роздача завантажених зображень із папки wwwroot/uploads)
* CORS Middleware (налаштовані політики безпеки для міжпортової взаємодії з React)

## Основний функціонал

* Авторизація та реєстрація: Форма входу та створення нового облікового запису користувача.
* Інтерактивне завантаження (Drag and Drop): Модуль UploadZone на базі HTML5 API, який дозволяє перетягувати графічні файли безпосередньо з комп'ютера у вікно браузера.
* Адаптивна сітка зображень: Компонент GalleryGrid динамічно підлаштовує кількість колонок під будь-які екрани (від мобільних телефонів до Full HD моніторів).
* Іммерсивний Lightbox: Повноекранний перегляд картинок у високій роздільній здатності із циклічним перемиканням та підтримкою гарячих клавіш клавіатури (Стрілки, Escape).
* Динамічна зміна теми: Перемикання між світлим та темним режимами інтерфейсу з автоматичним збереженням вибору користувача у localStorage.
* Висока швидкість обробки даних: Оптимізовано HTTP-кешування статичних файлів на сервері (повернення статусів 304 Not Modified), що знижує час відповіді до мінімуму.

## Структура репозиторію

* /practice: клієнтський SPA-додаток на React + TypeScript.
  * /src/components: ізольовані UI-компоненти (Header, Sidebar, GalleryGrid, LightboxModal, UploadZone).
  * /src/context: контекст для динамічної зміни візуальних тем.
  * /src/types: описи TypeScript інтерфейсів для моделей даних (ImageDto).
* /wwwroot/uploads: директорія сервера для фізичного зберігання завантажених медіафайлів.
* Program.cs: головна точка входу .NET додатка, налаштування CORS та ендпоінтів API.
* проект.csproj: конфіфікаційний файл C# проєкту.
